Portfolio

Three projects, one strategy

Each project maps to a specific role at Apple or NVIDIA. Together they form a portfolio that no pure-CS candidate can match.

Days 41–50

ML cache replacement predictor

→ Apple on-device memory · NVIDIA GPU cache hierarchy

LSTM trained on Champsim memory traces. Beats LRU baseline on SPEC CPU2017 benchmarks. Includes IEEE-format report, SHAP analysis, and quantization deployment study.

August

CUDA GEMM kernel optimization

→ NVIDIA GPU Architecture Engineer JD

Hand-written CUDA matrix multiplication kernels, progressively optimized. Reach 70–80% of cuBLAS performance. Profile with Nsight Compute, document roofline analysis per kernel.

October

ML-assisted RTL bug detection

→ Apple CPU Verification ML Tool Engineer

Classifier on RISC-V simulation traces predicting coverage gaps. Combines RTL/SystemVerilog background with ML — almost no candidates have this combination.
